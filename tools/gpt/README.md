# /tools/gpt – NI Open-Source Governance GPT

This folder contains configuration, documentation, and launch materials for the official GPT trained on NI's open-source governance files.

## 🔍 Purpose

The GPT helps all stakeholders — including NI leadership, SteerCo members, and LabVIEW contributors — understand and apply the official governance model.

## 📂 Files

- `ni-open-source-gpt-config-updated.json` — Complete prompt and instruction config
- `ni-gpt-optional-enhancements.md` — Branding (name, icon, welcome message)
- `ni-open-source-gpt-audit-log.md` — Transparency audit log of input sources and behavior
- `gpt-launch-announcement.md` — Copy-paste launch content for Discord and GitHub
