# Glossary

## CI/CD

Continuous Integration and Continuous Delivery, automated practices for building, testing, and delivering software.

## CI

Continuous Integration, the practice of frequently merging code changes and running automated tests to detect issues early.

## VIPC

VI Package Configuration, a container file that lists LabVIEW packages to install.

## PPL

Packed Project Library, a precompiled LabVIEW library used for distributing compiled VIs.
